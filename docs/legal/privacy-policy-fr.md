## 3. Données que nous collectons

### 3.1 Données traitées localement (sur votre appareil)

Ces données **ne quittent jamais votre téléphone** :

| Donnée | Finalité | Conservation |
|--------|----------|--------------|
| Votre voix | Transcription en texte | Non stockée (traitement en temps réel) |
| Transcription | Exécution de vos commandes | Durée de la session |
| Notifications | Lecture à votre demande | Non stockées par Diva |
| Contacts | Envoi de messages | Accès en lecture seule |
| Calendrier | Création de rappels | Accès en lecture/écriture |

### 3.2 Données envoyées au cloud (optionnel)

Si vous activez le traitement cloud pour les requêtes complexes :

| Donnée | Finalité | Conservation |
|--------|----------|--------------|
| Transcription anonymisée | Compréhension avancée | 0 (éphémère, non stockée) |
| Contexte conversation | Continuité | 24 heures maximum |

**Vous pouvez désactiver le traitement cloud** dans les paramètres de l'application. Diva fonctionnera en mode 100% local avec des capacités réduites.

### 3.3 Données de compte

| Donnée | Finalité | Conservation |
|--------|----------|--------------|
| Email | Identification | Jusqu'à suppression du compte |
| Préférences | Personnalisation | Jusqu'à suppression du compte |

## 4. Comment nous utilisons vos données

### 4.1 Finalités du traitement

| Finalité | Base légale (RGPD Art. 6) |
|----------|---------------------------|
| Exécuter vos commandes vocales | Consentement |
| Personnaliser votre expérience | Consentement |
| Assurer le fonctionnement technique | Intérêt légitime |
| Améliorer l'application (données agrégées) | Intérêt légitime |

### 4.2 Ce que nous ne faisons PAS

- ❌ Nous ne vendons jamais vos données
- ❌ Nous ne stockons pas vos enregistrements vocaux
- ❌ Nous n'utilisons pas vos données pour de la publicité ciblée
- ❌ Nous ne créons pas de profil comportemental
- ❌ Nous ne partageons pas vos données avec des tiers marketing

## 6. Sécurité des données

Nous mettons en œuvre des mesures techniques et organisationnelles appropriées :

| Mesure | Description |
|--------|-------------|
| Chiffrement en transit | TLS 1.3 pour toutes les communications |
| Chiffrement au repos | AES-256 pour les données stockées |
| Authentification | Tokens JWT avec expiration courte |
| Architecture local-first | ~80% du traitement sur votre appareil |
| Pas de logs vocaux | Votre voix n'est jamais enregistrée |

## 7. Vos droits

En vertu du RGPD, vous disposez des droits suivants :

| Droit | Description | Comment l'exercer |
|-------|-------------|-------------------|
| **Accès** | Obtenir une copie de vos données | Paramètres > Mes données > Exporter |
| **Rectification** | Corriger vos données | Paramètres > Profil |
| **Effacement** | Supprimer vos données | Paramètres > Supprimer mon compte |
| **Limitation** | Suspendre le traitement | Paramètres > Mode pause |
| **Portabilité** | Récupérer vos données | Paramètres > Mes données > Exporter |
| **Opposition** | Refuser certains traitements | Paramètres > Confidentialité |
| **Retrait consentement** | Annuler votre consentement | Paramètres > Permissions |

Pour exercer ces droits par email : privacy@[domain].com

Nous répondrons dans un délai de 30 jours.

## 8. Conservation des données

| Type de donnée | Durée de conservation |
